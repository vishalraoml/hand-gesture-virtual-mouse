# Hand Gesture Virtual Mouse

## Overview
A Computer Vision based virtual mouse developed using Python, OpenCV, MediaPipe, and PyAutoGUI. The system allows users to control mouse movements and perform mouse actions using hand gestures in real time.

## Features
- Real-time hand tracking
- Cursor movement using hand gestures
- Left click and right click functionality
- Scrolling using finger gestures
- Touch-free human-computer interaction
- Smooth and responsive cursor control

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## How to Run
1. Install the required libraries:
pip install opencv-python mediapipe pyautogui

2. Run the Python file:
python main.py
